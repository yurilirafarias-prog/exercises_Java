# 📚 Lista de Exercícios em Java #1

Este repositório contém uma série de exercícios básicos desenvolvidos em Java, com foco em lógica de programação e manipulação de dados.

Os exercícios foram implementados em um único arquivo (`App.java`), onde cada método representa um problema resolvido.

---

## 🚀 Objetivo

Praticar os fundamentos da programação utilizando Java, como:

* Entrada de dados com `Scanner`
* Operações matemáticas
* Manipulação de variáveis
* Organização de código em métodos
* Lógica de programação básica

---

## 📂 Estrutura do Projeto

```java
App.java
```

* O método `main` executa todos os exercícios automaticamente
* Cada exercício está separado em métodos (`ex01` até `ex09`)

---

## 🧠 Exercícios Implementados

### 🔹 EX01 - Saudação

Solicita o nome do usuário e exibe uma mensagem de boas-vindas.

---

### 🔹 EX02 - Operações Matemáticas

Realiza:

* Soma
* Subtração
* Multiplicação
* Divisão

---

### 🔹 EX03 - Cálculo de IMC

Calcula o Índice de Massa Corporal com base no peso e altura.

---

### 🔹 EX04 - Aumento Salarial

Calcula o novo salário com aumento de 15%.

---

### 🔹 EX05 - Salário de Vendedor

Calcula o salário final considerando:

* Salário base
* Comissão por produto
* Percentual sobre vendas

---

### 🔹 EX06 - Troca de Valores

Troca os valores de duas variáveis utilizando uma variável auxiliar.

---

### 🔹 EX07 - Número Invertido

Inverte um número inteiro de 4 dígitos.

---

### 🔹 EX08 - Consumo Médio

Calcula o consumo médio de combustível de um veículo.

---

### 🔹 EX09 - Conversão de Tempo

Converte horas e minutos para segundos e exibe o total.

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/yurilirafarias-prog/exercises_java.git
```

2. Abra em uma IDE (IntelliJ, VS Code, etc.)

3. Execute a classe `App.java`

---

## ⚠️ Observações

* Todos os exercícios utilizam entrada de dados via terminal
* O programa executa todos os exercícios em sequência
* É necessário inserir os dados conforme solicitado no console
